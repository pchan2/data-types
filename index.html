Basic JavaScript Data Types

# Type Description
1 undefined When data is undefined, a variable is declared without an assigned argument.

Example: let x;
2 Boolean Boolean presents two values: true or false. Boolean values can also be written as 1 or 0 is useful in if statements or for loops. when a Boolean conditionals determines what sections of code are executed.
3 Number Number comes in three forms: integer (whole number), float (decimals) and NaN (Not a Number). The maximum value of a Number type is Number.MAX_SAFE_INTEGER which returns 9007199254740991 while the minimum value is Number.MIN_SAFE_INTEGER which returns -9007199254740991. This means anything beyond the maximum or minimum value are unreliable and can result in inaccurate results.
4 String String is textual data. Each character inside the string occupies a position known as the index number which starts with 0, followed by 1, 2 3 and so on. Two Strings can be joined up or concatenated using + or String.concat().
5 BigInt BigInt stands for Big Integer and therefore is not a decimal. It can be constructed in two ways: by adding an n to a number or passing a number in BigInt() It allows one to safely add, subtract, multiply and divide the maximum or minimum value.

Example: BigInt(Number.MAX_SAFE_INTEGER) + 2n will return 9007199254740993n.
6 Symbol A Symbol is a unique value which is usually used for creating unique property keys in objects. To make a new Symbol, the syntax is Symbol() and a string can be passed in to identify what the Symbol is but this is optional and will not have an impact on the unqiue value because "under the hood" a unique number value is generated.

Example: const symb1 = Symbol();
7 Object The Object class stores keyed collections. Object properties can be created, retrieved, updated and deleted

Example: const car = {
            wheels: "alloy"
            gear: "auto"
            colour: "white"
            seats: 5
          };
8 Function A Function when invoked will run an operation. It can contain conditionals by using the IF statement. It can be given a name or be anonymous. It can contain parameters which acts as placeholders for arguments to be passed in.
